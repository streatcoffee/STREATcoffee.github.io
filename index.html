Current Project - SitDown

SitDown is a web application that assess the how congested certain common areas are within different buildings of the RMIT campus. A microcontroller will be placed in study areas that receive a high amount of foot traffic. Probe packages will be sent from WIFI enabled devices while trying connect to a WIFI network. These probe packages will be received via the microcontroller. On request, the application will push data to the user based on how close they are to an available seat. If no seats are available in that area the user will be able to give feedback on the prediction made by the application.
